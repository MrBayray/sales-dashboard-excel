📊 Sales Dashboard (Excel Project)

## 📌 Overview
This project is a simple Sales Dashboard built using Microsoft Excel. It analyzes sales data to provide insights into product performance and overall sales trends.

---

## 🎯 Objectives
- Calculate key performance indicators (KPIs)
- Analyze sales by product
- Visualize data using charts
- Present insights in a clean dashboard layout

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Column Chart)

---

## 📈 Key Metrics
- Total Sales
- Average Sales
- Number of Transactions

---

## 📊 Analysis Performed
A pivot table was used to group sales data by product and calculate total sales for each category. This helped identify the best-performing product.
